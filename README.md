# NPCI_Analysis
NPCI_Analysis is a data analytics project analyzing India’s digital payment ecosystem using Python, PostgreSQL, and Tableau. It explores transaction trends, growth patterns, and market share across systems like UPI, IMPS, AePS, and NACH.

# Objectives
Analyze transaction trends over time
Compare payment systems (UPI, IMPS, AePS, NACH, NETC)
Identify market share distribution
Study growth patterns and seasonal variations
